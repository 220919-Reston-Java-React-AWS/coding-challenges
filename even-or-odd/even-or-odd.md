## Duration
30 minutes

## Challenge
Write a conditional block that prints "Even" if the number passed is even and "Odd" if the number is odd.
  
## Examples
```java
//Sample Input
int number = 1156;

//Expected Output
Even
```

```java
//Sample Input
int number = 2433;

//Expected Output
Odd
```

**HINT:** look into the % operator
